    Setup & Sharing

    Install

    On iPhone: open in Safari, tap Share → Add to Home Screen.

    Sharing to this app

    Android / Chrome: works out of the box — appears in the system share sheet after install.

    iPhone: Safari doesn't yet expose PWAs as share targets. Use the built-in Shortcuts app:

    1. Shortcuts → + → Add Action → URL.
    2. Set URL to:
    3. Add Open URLs action below it.
    4. In the shortcut details, enable Show in Share Sheet; accept URLs and Text.
    5. Edit the URL action: replace the placeholder with a Shortcut Input magic variable (URL-encoded).

    A ready-to-use template is in the repo's README.md.
